# Counter Scam Agency

Counter Scam Agency 是一款以防範詐騙為主題的角色扮演遊戲（RPG）。玩家扮演「反詐情報官」，在城市中協助市民解析詐術、阻止匯款、追查詐團，並逐步解鎖調查工具與技能。

## 目標
- 以遊戲化方式傳遞防詐知識。
- 保持劇情與案件推理的沉浸感。
- 打造可延伸的系統架構，方便快速迭代內容。

## 特色
- 案件調查循環：蒐集資訊 → 判讀 → 處置 → 回饋。
- 技能與工具養成：社交洞察、金融偵測、數據取證、心理韌性等技能樹。
- 詐術圖鑑：記錄已破解的詐騙類型與關鍵徵兆，作為教育資源。
- 城市安全度：玩家表現會改變城市局勢與事件難度。
